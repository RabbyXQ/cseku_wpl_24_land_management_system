First create .env file at the cloned root directory
```
DATABASE_URL="mysql://user:pass@host:port/databasename"
DB_HOST=
DB_USER=
DB_PASSWORD=
JWT_SECRET=
```

then 

```
npm install
npm run dev
```

To build
```
npm build
npm start
```

You also can use other package managers.
